# PS5 Pro — Premium Showcase

A premium, fan-made showcase website for the PlayStation 5 Pro console featuring the DualSense Edge controller and upcoming game titles.

## Features

- **PS5 Pro Section** — $999 limited console with full spec breakdown
- **DualSense Edge Controller** — Bundled with PS5 Pro, detailed feature list
- **Upcoming Games** — Cover art gallery for upcoming PS5 titles
- **Premium Design** — Dark theme with smooth animations and responsive layout

## Getting Started

```bash
npm start
```

Opens a local server at `http://localhost:3000` serving the `src/` directory.
